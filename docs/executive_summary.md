# Executive Summary

This project is an AI inference optimization and deployment decision engine.

It evaluates deployment configurations across:

- latency
- accuracy
- energy
- carbon emissions
- monthly cost
- queueing delay
- autoscaling pressure
- deployment mode
- optimization strategy

## Key Value

Most ML projects stop at accuracy. This project extends the analysis into production deployment decisions.

It demonstrates how different deployment objectives produce different optimal configurations:

- fastest
- cheapest
- lowest carbon
- lowest energy
- balanced
